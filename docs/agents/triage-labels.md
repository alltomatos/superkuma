# Triage Labels

As skills falam em termos de cinco papéis canônicos de triagem. Esta tabela mapeia esses papéis para as labels reais usadas neste tracker.

| Papel canônico    | Label no nosso tracker | Significado                                |
| ----------------- | ---------------------- | ------------------------------------------ |
| `needs-triage`    | `needs-triage`         | Mantenedor precisa avaliar a issue         |
| `needs-info`      | `needs-info`           | Aguardando mais informação do reporter     |
| `ready-for-agent` | `ready-for-agent`      | Totalmente especificada, pronta p/ agente  |
| `ready-for-human` | `ready-for-human`      | Requer implementação humana                |
| `wontfix`         | `wontfix`              | Não será tratada                           |

Edite a coluna da direita para casar com o vocabulário que você realmente usa.
